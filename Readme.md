# plugin for chrome
---
## 描述
- 这是针对 chrome 制作的 一些小型网站插件
## 使用说明
- git 本文件地址 到你本机上
- chrome 浏览器 打开插件面板
- 拖拽 这个 git 中任何一个文件夹到 chrome 插件面板即可使用
## 插件说明
- 九酷
  是九酷播放音乐页的音乐src，针对下载 MP3 的地址
